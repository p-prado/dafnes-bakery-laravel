### Funcionalidad
Este proyecto de Laravel es el sitio web de una panadería/pastelería, Dafne's Bakery. Se muestran algunas páginas de categoría con sus productos correspondientes. En la página Admin Dashboard, se pueden gestionar los productos y las categorías.
### Acerca de las imágenes
En el archivo ```DDL.sql``` se declara un valor predeterminado para las imagenes, ya que la carga de imagenes todavía no está admitida en este código.
